# Neon Ark Warden Onboarding System

**Assignment Deliverables:**

- neon_ark_erd.txt → ER Diagram (text version using DBML)
- neon_ark.sql → Complete PostgreSQL script

This design supports every menu in the assignment:

- Add New Warden
- View Wardens (All / By ID / By Status / By Role)
- Update Warden fields
- Manage Certifications (Add, View, Mark Expired, Remove)
- Deactivate / Terminate Warden

Uses lookup tables for consistency and proper constraints for Warden identity.
